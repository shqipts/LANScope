# Changelog

## v0.1.0

Initial private MVP release.

### Added
- SwiftUI 3-tab app structure: Info / LAN / Tools
- Wi‑Fi/network info view
- limited cellular/SIM info section
- Local Network + Location onboarding flow
- LAN scanner with stronger default discovery behavior
- Bonjour/mDNS discovery integration
- host caching and repeat-scan merge logic
- host details, share/export actions
- ping, DNS, WHOIS/RDAP, and port scan tools
- private GitHub release with unsigned IPA artifact

### Notes
- unsigned IPA is for artifact/testing workflows, not direct normal iPhone install
- iOS networking restrictions still limit how much a scanner can discover compared with desktop tools
