# Changelog

## v0.1.0

Initial public MVP release.

### Added
- SwiftUI 3-tab app structure: Info / LAN / Tools
- Wi‑Fi / network info view
- limited cellular/mobile-data info section
- first-launch onboarding for Local Network and Location permissions
- stronger default LAN scanning behavior
- Bonjour / mDNS discovery integration
- host caching and repeat-scan merge logic
- host details, share/export actions
- ping, DNS, WHOIS / RDAP, and port scan tools
- GitHub release with unsigned IPA artifact

### Notes
- unsigned IPA is for artifact/testing workflows, not direct normal iPhone installation
- iOS networking restrictions still limit how much a scanner can discover compared with desktop tools
